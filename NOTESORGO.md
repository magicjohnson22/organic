<h2 id="week-1-—-heteroatom-chemistry-boron-i-foundations--hydroboration">Week 1 — Heteroatom Chemistry: <strong>BORON I (Foundations + Hydroboration)</strong></h2>
<h3 id="core-concepts-boron-fundamentals">Core concepts (Boron fundamentals)</h3>
<ul>
<li>
<p>☐ Boron is <strong>electron-deficient</strong> (empty p orbital), acts as a <strong>Lewis acid</strong></p>
</li>
<li>
<p>☐ <strong>Trigonal planar (3-coordinate)</strong> boranes vs <strong>tetrahedral (4-coordinate)</strong> borates (“ate” complexes)</p>
</li>
<li>
<p>☐ B–H bonds are hydridic relative to boron; polarity concepts</p>
</li>
<li>
<p>☐ Why boron reagents are selective (sterics + electronics)</p>
</li>
<li>
<p>☐ Reagent recognition and what each is used for:</p>
<ul>
<li>
<p>☐ <strong>BH₃·THF</strong>, <strong>BH₃·SMe₂</strong></p>
</li>
<li>
<p>☐ <strong>9-BBN</strong> (bulky borane)</p>
</li>
<li>
<p>☐ <strong>catecholborane</strong> (often for alkynes / vinylboranes)</p>
</li>
</ul>
</li>
</ul>
<h3 id="reactions--mechanisms-must-be-drawable--explainable">Reactions &amp; mechanisms (must be drawable + explainable)</h3>
<ul>
<li>
<p>☐ <strong>1a</strong> Hydroboration of alkenes: <strong>concerted 4-centre transition state</strong></p>
</li>
<li>
<p>☐ <strong>Regioselectivity:</strong> boron adds to <strong>less substituted carbon</strong> (anti-Markovnikov outcome after oxidation)</p>
</li>
<li>
<p>☐ <strong>Stereoselectivity:</strong> <strong>syn addition</strong> of H and B across the double bond</p>
</li>
<li>
<p>☐ Explain (in words) WHY:</p>
<ul>
<li>
<p>☐ steric control (bulky B goes to less substituted carbon)</p>
</li>
<li>
<p>☐ electronic control (TS polarization; developing positive charge distribution concept)</p>
</li>
<li>
<p>☐ concerted → syn</p>
</li>
</ul>
</li>
</ul>
<h3 id="exam-tasks-you-must-be-able-to-do">Exam tasks you must be able to do</h3>
<ul>
<li>
<p>☐ Given an alkene + borane reagents → predict product(s) correctly</p>
</li>
<li>
<p>☐ Show correct stereochemistry for cyclic and acyclic alkenes</p>
</li>
<li>
<p>☐ Explain anti-Markovnikov + syn in mark-scheme language</p>
</li>
</ul>
<p><strong>Examinability: 9/10</strong></p>

<h2 id="week-2-—-heteroatom-chemistry-boron-ii-hydroboration–oxidation--organoborane-logic">Week 2 — Heteroatom Chemistry: <strong>BORON II (Hydroboration–Oxidation + Organoborane Logic)</strong></h2>
<h3 id="hydroboration–oxidation-core">Hydroboration–oxidation (core)</h3>
<ul>
<li>
<p>☐ <strong>2a</strong> Oxidation mechanism: H₂O₂/NaOH forms peroxy anion → attacks boron → borate intermediate</p>
</li>
<li>
<p>☐ <strong>Key step:</strong> <strong>1,2-migration (alkyl shift from B to O)</strong> with <strong>retention of configuration</strong></p>
</li>
<li>
<p>☐ Final product: alcohol at less substituted carbon (<strong>anti-Markovnikov</strong>) with stereospecific outcome</p>
</li>
</ul>
<h3 id="organoborane-“follow-up”-transformations-module-style-family">Organoborane “follow-up” transformations (module-style family)</h3>
<p>You must be able to <strong>recognise these as C–B → functional group conversions</strong>, know stereochemical trends, and apply them:</p>
<ul>
<li>
<p>☐ <strong>3a</strong> Halogenation (Br₂/I₂) via boronate/“ate” complex → C–X (often stereospecific; can be treated as inversion-type in many teaching schemes)</p>
</li>
<li>
<p>☐ <strong>3b</strong> Chlorination conditions that proceed via radical behaviour → loss of stereospecificity (conceptual)</p>
</li>
<li>
<p>☐ <strong>4a</strong> Protonolysis (acid) → C–H (alkane); stereochemical note (often retained)</p>
</li>
<li>
<p>☐ <strong>5a</strong> Amination routes (organoborane → amine) (pattern recognition + outcome)</p>
</li>
<li>
<p>☐ <strong>6a/6b/6c</strong> Carbonylation of trialkylboranes (CO insertion logic) → aldehydes / ketones / tertiary alcohols depending on pathway and workup</p>
</li>
<li>
<p>☐ <strong>7a</strong> Alkyne hydroboration (esp. catecholborane) → vinylboranes/vinylboronates (cis addition logic; used as coupling handles)</p>
</li>
</ul>
<h3 id="exam-tasks-you-must-be-able-to-do-1">Exam tasks you must be able to do</h3>
<ul>
<li>
<p>☐ Given reagents → select correct conversion family (C–B→C–O/C–X/C–N/C–H)</p>
</li>
<li>
<p>☐ Predict product(s) + any stereochemical consequences</p>
</li>
<li>
<p>☐ Link vinylboron chemistry to Suzuki coupling logic (bridge concept)</p>
</li>
</ul>
<p><strong>Examinability: 7–9/10 (hydroboration–oxidation is 9/10; follow-ups vary by paper)</strong></p>
